Detailed Description

template<typename FUNCTOR_TYPE>
struct flecsi::execution::functor_task_wrapper__< FUNCTOR_TYPE >

The functor_task_wrapper__ type provides registation and execution callback functions for functor tasks. User tasks must be expressed as functor types that implement an execute method.

FUNCTOR_TYPEThe functor type of the functor task.
